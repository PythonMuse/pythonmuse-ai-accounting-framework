# Recommended VS Code Extensions for Finance Professionals

*A curated list of extensions that make VS Code more useful for accounting and finance work.*

---

## Essential Extensions

### Python (Microsoft)

**What it does:** Adds Python language support -- syntax highlighting, code completion, debugging, and the ability to run scripts directly.

**Why it matters for accounting:** If you are running any Python scripts (data analysis, reconciliation, reporting), this is required.

**Install:** Search "Python" in the Extensions panel. Publisher: Microsoft.

---

### Markdown Preview Enhanced

**What it does:** Provides a rich preview of Markdown files with support for tables, diagrams, and export to PDF.

**Why it matters for accounting:** You will write a lot of Markdown (plans, status updates, workpapers, CLAUDE.md). This extension lets you see the formatted version side by side with your source.

**Install:** Search "Markdown Preview Enhanced" in the Extensions panel.

---

### Rainbow CSV

**What it does:** Color-codes CSV columns so you can visually distinguish fields without opening the file in Excel.

**Why it matters for accounting:** When you are working with bank statements, GL exports, or trial balance files in CSV format, this makes it much easier to scan the data quickly.

**Install:** Search "Rainbow CSV" in the Extensions panel.

---

## AI Extensions (Choose One)

### Claude (Anthropic)

**What it does:** Integrates Claude directly into VS Code as an AI assistant that can read your project files, write code, and execute scripts.

**Why it matters for accounting:** Claude can help with data analysis, script generation, documentation, and workflow automation -- all within your project context.

**Install:** Search "Claude" in the Extensions panel. [Claude Code](https://claude.ai/code) is also available as a terminal-based tool.

---

### GitHub Copilot

**What it does:** AI-powered code suggestions and chat within VS Code.

**Why it matters for accounting:** Copilot can help complete Python scripts, explain code, and suggest improvements as you work.

**Install:** Search "GitHub Copilot" in the Extensions panel. Requires a GitHub account with Copilot access.

---

## Useful Additions

### Excel Viewer

**What it does:** Lets you view Excel and CSV files in a table format directly inside VS Code.

**Why it matters for accounting:** Quick data inspection without switching to Excel. Useful for verifying source files.

**Install:** Search "Excel Viewer" in the Extensions panel.

---

### GitLens

**What it does:** Enhances the built-in Git support with blame annotations, file history, and comparison tools.

**Why it matters for accounting:** See who changed what and when -- directly in the editor. This supports your audit trail and version control practices.

**Install:** Search "GitLens" in the Extensions panel.

---

### vscode-pdf

**What it does:** View PDF files directly inside VS Code.

**Why it matters for accounting:** Quickly reference a bank statement or vendor invoice without leaving your workspace.

**Install:** Search "vscode-pdf" in the Extensions panel.

---

## Installation Steps

1. Open VS Code
2. Click the Extensions icon in the left sidebar (four squares icon)
3. Type the extension name in the search bar
4. Click "Install"
5. Some extensions may require a restart of VS Code

Start with Python and Markdown Preview Enhanced. Add others as your workflow develops.
