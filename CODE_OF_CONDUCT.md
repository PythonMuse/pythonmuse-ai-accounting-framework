# Code of Conduct

## Our Commitment

The PythonMuse AI Accounting Framework is a professional community for accounting and finance professionals learning to work with AI tools. We are committed to providing a welcoming, respectful environment for everyone -- regardless of experience level, background, or role.

## Expected Behavior

- **Be professional.** This is a community of finance professionals. Communicate the way you would with a respected colleague.
- **Be constructive.** If you see something that could be improved, suggest a solution. Critique the work, not the person.
- **Be patient.** Many contributors are new to Git, Python, or open source. Help them learn.
- **Be honest.** If you are unsure about something, say so. Accounting is built on accuracy, and so is this project.

## Unacceptable Behavior

- Personal attacks, insults, or derogatory comments
- Harassment of any kind
- Sharing others' private information
- Posting real financial data, client information, or proprietary materials
- Dismissing someone's contribution because of their experience level

## Scope

This Code of Conduct applies to all project spaces -- including GitHub Issues, Pull Requests, discussions, and any related communications.

## Reporting

If you experience or witness unacceptable behavior, please contact the project maintainers at the email listed on the [PythonMuse GitHub profile](https://github.com/PythonMuse).

Reports will be reviewed promptly and handled confidentially.

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this Code of Conduct. Repeated violations may result in being blocked from the project.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1, with modifications for a professional accounting audience.
